## 🚀 Quick Start

### Prerequisites
- Java 11 or higher
- Maven 3.6+
- IDE: IntelliJ IDEA
- Database: H2 (embedded) or MySQL

### Setup Steps

1. **Clone/Create Project**
   ```bash
   cd spring-boot
   ```

2. **Build Project**
   ```bash
   mvn clean install
   ```

3. **Run Application**
   ```bash
   mvn spring-boot:run
   ```

4. **Access Application**
   - Base URL: `http://localhost:8080`
   - Project 1 H2 Console: `http://localhost:8080/h2-console`

## 📋 API Endpoints & Pages

### **PROJECT 1 - Order Management REST APIs**

| Method | Endpoint | Description |
|--------|----------|-------------|
| POST | `/api/orders` | Create new order |
| GET | `/api/orders` | Get all orders |
| GET | `/api/orders/{id}` | Get order by ID |
| GET | `/api/orders/number/{orderNumber}` | Get by order number |
| GET | `/api/orders/customer/{customerId}` | Get customer's orders |
| GET | `/api/orders/status/{status}` | Get orders by status |
| GET | `/api/orders/search?customerName=...` | Search by name |
| PUT | `/api/orders/{id}` | Update order |
| PATCH | `/api/orders/{id}/status` | Update status only |
| DELETE | `/api/orders/{id}` | Delete order |
| GET | `/api/orders/stats/total` | Get total count |

### **PROJECT 2 - Book Store Pages & APIs**

**Pages (Thymeleaf):**
- `GET /` - Home page with featured books
- `GET /login` - Login form
- `GET /register` - Registration form
- `GET /catalog` - Book catalog with search & filters

**REST APIs:**
- `POST /api/auth/login` - User login
- `POST /register` - User registration
- `GET /api/books` - Get all books
- `GET /api/books/{id}` - Get book by ID
- `GET /api/books/search?keyword=...` - Search books
- `GET /api/books/category/{category}` - Filter by category
- `GET /api/books/author/{author}` - Filter by author

## ⚙️ IntelliJ Copy-Paste Setup (Step by Step)

### **For PROJECT 1: Order Management**

1. **Create Maven Project**
   - File → New → Project
   - Select: Maven
   - GroupId: `com.order`
   - ArtifactId: `order-management-basic`
   - Version: `1.0-SNAPSHOT`
   - Java: 11

2. **Create Packages**
   - Right-click `src/main/java` → New → Package
   - Create: `com.order`

3. **Copy Java Files** (to `src/main/java/com/order/`)
   - Copy `order-entity.java` → paste into `com.order` package
   - Copy `order-repository.java` → paste into `com.order` package
   - Copy `order-service.java` → paste into `com.order` package
   - Copy `order-controller.java` → paste into `com.order` package
   - Copy `order-management-application.java` → paste into `com.order` package

4. **Copy Configuration**
   - Copy `pom.xml` → replace the generated one in project root
   - Copy `application.properties` → paste into `src/main/resources/`

5. **Reload Maven**
   - Right-click `pom.xml` → Maven → Reload Projects
   - Wait for dependencies to download

6. **Run Application**
   - Right-click `order-management-application.java` → Run
   - Or press Shift+F10
   - App runs on: `http://localhost:8080`
